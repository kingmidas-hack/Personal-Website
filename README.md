### ‼️ Prerequisites

This project uses `Node.js` and `npm` as package manager

```bash
  npm install -g npm
```
--------------------------------------------------------------------------------

Clone the project locally in your preferred directory
```
git clone https://github.com/kingmidas-hack/Personal-Website.git
```

<!-- Installation -->
### ⚙️ Installation

`cd` into project and install dependencies with npm

```bash
  cd Personal-Website && npm install
```
